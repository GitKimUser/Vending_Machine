# 자판기 미션

## 기능 목록

### 1. 도메인 모델 (핵심 로직)
- [x] 상품(Item) 클래스 구현 (상품명, 가격, 재고)
- [x] 상품 목록(Inventory) 클래스 구현 (상품 리스트 관리, 재고 확인, 차감)
- [x] 투입 금액(Money) 클래스 구현 (금액 저장, 10원 단위 검증)
- [ ] 잔돈(Change) 클래스 또는 Enum 구현 (동전 종류 및 계산 로직)
- [ ] 자판기(VendingMachine) 클래스 구현 (모든 객체를 조합해 핵심 기능 수행)
    - [ ] 상품 구매 기능
    - [ ] 잔돈 반환 기능

### 2. 입력 (InputView)
- [ ] 자판기가 보유할 상품 정보(상품명, 가격, 수량) 입력
- [ ] 사용자가 투입할 금액 입력
- [ ] 사용자가 구매할 상품명 입력

### 3. 출력 (OutputView)
- [ ] 투입 금액 출력
- [ ] 잔돈 반환 내역 출력 (동전별 개수)
- [ ] *[선택]* 상품 구매 후 남은 재고 목록 출력

### 4. 예외 처리 (Validation)
- [x] [ERROR] 상품 입력 형식이 올바르지 않은 경우 (가격이 10원 단위가 아니거나 100원 미만인 경우 등)
- [x] [ERROR] 투입 금액이 숫자가 아니거나 10원 단위가 아닌 경우
- [x] [ERROR] 존재하지 않는 상품명을 입력한 경우
- [ ] [ERROR] 상품의 재고가 부족한 경우
- [ ] [ERROR] 투입 금액이 상품 가격보다 부족한 경우
